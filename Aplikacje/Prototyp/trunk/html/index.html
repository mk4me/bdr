This is version 1.3 (30.09.2010).
To see our planned functionality have a look at our roadmap.
In case of problems consult our known issues page, please. There are some workarounds for the most well known and common problems.

This application requires Java JRE 1.6.0 (click for download and install)
If Java is already installed on your computer choose one of the two possible launchers below.
1. Web Browser launcher.

a) Just press the button and if you previously accepted our signed applet you should see a login dialog.
b) Enter your login name, password and domain and start using the client

Application will be embeded in the internet browser window. This option requires the browser to be correctly configured to run java applets. Also browser window has to be openned all the time if the Motion Web Client is in use.
2. Java Web Start launcher.

a) Clik on the link. You will be prompted to run a jnlp Java Web Start Application. Confirm.
b) After a short download you will be prompted to run a signed application. Choose "Run"
c) You will see a login dialog. Enter your login name, password and domain and start using the client.

This option is independent of the internet browser and requires only Java to be installed. You may also close the browser after the application is started. You may also download MotionClientApplication.jnlp file, save it somewhere in your home directory and use for launching the Web Client in the future.
