Welcome to Motion Web Client Launcher page!

This application requires Java JRE 1.6.0 (click here to download and install)

If Java is already installed on your computer choose one of the two possible launchers below.
1. Web Browser launcher.

Application will be embeded in the internet browser window. This option requires the browser to be correctly configured to run java applets. Also browser window has to be openned all the time if the Motion Web Client is in use.
2. Java Web Start launcher.

This option is independent of the internet browser and requires only Java to be installed. You may also close the browser after the application is started. You may also download MotionClientApplication.jnlp file, save it somewhere in your home directory and use for launching the Web Client in the future.
