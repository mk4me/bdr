The root package for motion client application. It contains two subpackages:
  1. motion.database - server connection facade
  2. motion.appet - motion web client applet
This package also contains the only common structure for the two subpackages: externalized strings tools, Messages class, list of possible strings in file messages.properties with translation to Polish and its latin1 version for interoperability between different platforms in file messages-latin1.properties.
