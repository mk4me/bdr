Human Motion Database Web Service Interface

Welcome to the Human Motion Database Web Services!


This page contains the resources needed for connecting client-side software using the Human Motion Database (HMDB) functionality.

The access to the Web Service layer requires HTTP communication. Since this temporary deployment does not possess a trusted certificate, you will need to download and install the following root certificate among the trusted root CA on your machine (click for download and install).

If you need to create a new user account for your access to the HMDB functionality - please use this page to create account and activate it subsequently.
